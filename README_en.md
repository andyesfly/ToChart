## ✨ Features

### Multi-Source Connectivity
Connect to major databases directly — no middleware needed, data never leaves your machine.

| Data Source | Support | Notes |
|-------------|:-------:|-------|
| MySQL | ✅ | Remote & local instances, SSL encryption |
| PostgreSQL | ✅ | Full SSL support |
| Oracle | ✅ | Oracle DB & Oracle Cloud |
| SQL Server | ✅ | All versions supported |
| MariaDB | ✅ | MySQL protocol compatible |
| SQLite | ✅ | Local file-based DB, perfect for mobile & small projects |
| Excel | ✅ | `.xlsx` / `.xls` format support |
| CSV | ✅ | UTF-8 encoding, auto-delimiter detection |

> MongoDB, Snowflake and more coming soon.

### 30+ Chart Types
Over 30 professional chart types, drag-and-drop creation, real-time rendering.

- Bar, Line, Pie, Scatter, Bubble, Radar, Heatmap, Sankey, Treemap & more
- Drag-and-drop interface, minimal learning curve
- Auto color matching & layout optimization

### 100% Local Data Processing

```
Data Ingestion → Data Wrangling → Chart Rendering → Export
      ↓               ↓               ↓               ↓
    Local           Local           Local           Local
```

All data operations happen entirely on your device — **no third-party servers involved**, zero data leak risk. **GDPR compliant.**

### Smart SQL Editor
Built-in enhanced SQL editor for all skill levels.

- ✅ Syntax highlighting & auto-completion
- ✅ Real-time error checking & suggestions
- ✅ Multi-tab management
- ✅ SQL formatting

### One-Click Export
Export to PDF, images, and more with original layout and color retained.

## 🚀 Quick Start

1. **Download** → Visit [tochart.tech](https://tochart.tech) to download the desktop app
2. **Connect Data** → Choose your database or upload CSV/Excel files
3. **Drag & Drop** → Pick a chart type, drag fields into position
4. **Export & Share** → Export to PDF or images in one click

> No registration, no login — install and go.

## 🔒 Security

ToChart uses a localized architecture to ensure your data never leaves your device:

- Data is read, processed, and rendered locally — no cloud upload required
- Direct database connection — no intermediate proxy
- GDPR compliant data protection
- Device-bound accounts, fully transparent and controllable
