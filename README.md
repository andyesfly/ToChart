## ✨ 功能特性

### 多数据源直连
直接连接主流数据库，无需中间层，数据始终留存在本地。

| 数据源 | 支持 | 说明 |
|--------|:----:|------|
| MySQL | ✅ | 支持远程与本地实例，SSL 加密连接 |
| PostgreSQL | ✅ | 安全高效，支持 SSL 连接 |
| Oracle | ✅ | 支持 Oracle DB 及 Oracle Cloud |
| SQL Server | ✅ | 全系列版本支持 |
| MariaDB | ✅ | 完全兼容 MySQL 协议 |
| SQLite | ✅ | 本地文件数据库，适合移动端/小型项目 |
| Excel | ✅ | 支持 `.xlsx` / `.xls` 格式 |
| CSV | ✅ | UTF-8 编码，自动分隔符识别 |

> MongoDB, Snowflake 等更多数据源正在筹备中。

### 30+ 可视化图表
提供超过 30 种专业图表类型，拖拽式操作，实时渲染。

- 柱状图、折线图、饼图、散点图、气泡图、雷达图、热力图、桑基图、树图…
- 拖拽式创建，学习成本极低
- 自动配色与布局优化

### 100% 本地化数据处理

```
数据读取 → 数据整理 → 图表渲染 → 导出
   ↓          ↓           ↓         ↓
  本地       本地        本地      本地
```

所有数据操作均在用户设备上完成，**不经过任何第三方服务器**，杜绝数据泄露风险。符合 **GDPR** 合规标准。

### 智能 SQL 编辑器
内置增强型 SQL 编辑器，满足从入门到专家的不同需求。

- ✅ 语法高亮与自动补全
- ✅ 实时语法检查与错误提示
- ✅ 多标签页管理
- ✅ SQL 格式化

### 一键导出
支持一键导出为 PDF、图片等格式，保留原始排版与配色。

## 🚀 快速开始

1. **下载** → 访问 [tochart.tech](https://tochart.tech) 下载桌面客户端
2. **连接数据** → 选择你的数据库或上传 CSV/Excel 文件
3. **拖拽出图** → 选择图表类型，拖拽字段到对应位置
4. **分享导出** → 一键导出为 PDF 或图片

> 无需注册，无需登录，即装即用。
